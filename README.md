# python-data-analysis-project
Python project demonstrating data cleaning, analysis, and ML concepts

This project demonstrates a simple end-to-end data analysis workflow using Python and pandas.

## What this project does
- Loads a CSV dataset
- Performs basic data inspection
- Checks for missing values
- Computes summary statistics
- Calculates average income grouped by a target variable

## Technologies used
- Python
- pandas
- GitHub Actions (for automated execution)

## Dataset
The dataset is a small sample containing:
- age
- income
- years of experience
- target (binary outcome)
